Mes projets

Qovery Engine est une bibliothèque de couches d’abstraction open-source qui facilite le déploiement d’applications sur AWS, GCP, Azure et d’autres fournisseurs de Cloud en quelques minutes. Le Qovery Engine est écrit en Rust et utilise Terraform, Helm, Kubectl et Docker pour gérer les ressources. J'ai réalisé ce projet de manière personel avec HTML5, CSS3 et JavaScript.
